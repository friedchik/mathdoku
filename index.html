Mathdoku

    Hello there! This is my Mathdoku game where it consists with numbers to enter in the box

    A sudoku puzzle begins with a grid in which some of the numbers are already in place

    Each game should have one and only solution to be considered a valid Mathdoku game

    There will be no number that is repeated throughout the game and should be completed numbers from 1 to 9

    Players must click the number in order to finish the game
